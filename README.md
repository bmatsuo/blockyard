Install
=======

    git clone git@github.com:bmatsuo/blockyard
    cd blockyard
    ./build.sh

Usage
=====

    ./bin/blockd

Docs
====

    ./docs.sh [ -http=LADDR | IMPORT TAG ]

Examples
--------

    ./docs.sh -http=:6060
    ./docs.sh schuntil/log NewSyslog
